Setup Guide

1. Add the addon

Install the addon by adding Pantry Tracker to custom repositories or use the link below and start the addon.

2. Add the custom integration

Add the integration Pantry Tracker Sensor to custom repositories or use the link below.

3. Restart HomeAssistant

 

4. Configure the Integration

Navigate to Settings > Devices > Add Integration > Pantry Tracker Sensor 

 

 

Name: Default Usage
Update Interval:  30 Time in seconds to poll for new data
Pantry Tracker Host:  127.0.0.1 IP of host where Pantry Tracker is running
Pantry Tracker Port: 8099 Port the Pantry Tracker is running on
API Key:  N/A Enter the API Key obtained from Pantry Tracker
