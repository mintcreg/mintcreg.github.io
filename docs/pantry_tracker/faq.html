Frequently Asked Questions

1. How often will the project be updated?

This is something I do in my spare time and therefore cannot guarantee any timescales on when updates will be provided

2. How can I contribute to the project?

Feel free to fork the repository, make your changes, and submit a pull request.

3. I want to see XYZ can you implement it?

Feel free to submit a pull request for feature enhancements, however like #1 there is no guarantees to timescale or feature implementation

3. Entities no longer showing

Usually this means that the API key has changed, you can reconfigure this by visiting Settings > Integrations > Pantry Tracker Sensor > Configure (Enter new API key and click submit)
